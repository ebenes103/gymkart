#  GymKart - E-commerce Platform for Gym Equipment

GymKart is a comprehensive e-commerce platform built with Django that connects gym equipment sellers with fitness enthusiasts. Sellers can list their products, and buyers can browse, wishlist, and purchase equipment.

![GymKart Logo](https://via.placeholder.com/200x100/DC2626/FFFFFF?text=GYMKART)

##  Features

### For Buyers

 **Browse Products** - View all available gym equipment
 **Search Functionality** - Find products by name, brand, or category
 **Category Filtering** - Filter products by Weights, Cardio, and Accessories
 **Wishlist** - Save favorite products for later
 **Shopping Cart** - Add products with weight options
 **AI Chatbot** - 24/7 customer support for product inquiries
 **Newsletter Subscription** - Stay updated on new products and offers

### For Sellers

**Seller Dashboard** - Manage your products inventory
**Add Products** - List new products with multiple weight options
**Edit Products** - Update product details and stock
**Stock Management** - Track inventory per weight option
**Low Stock Alerts** - Get notified when stock is running low

### For Admins

**User Management** - Approve/reject seller applications
**Dashboard** - View platform statistics
**Newsletter Management** - View subscriber emails

## 🛠️ Technology Stack

**Backend**: Django 6.0.2
**Frontend**: HTML5, Tailwind CSS, JavaScript
**Database**: SQLite (Development) / MySQL (Production)
**AI Chatbot**: tawk.to Integration
**Icons**: Font Awesome 6

## Prerequisites
